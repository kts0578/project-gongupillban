	<head></head>		:브라우저가 알아야 할 내용
	<title></title>		:문서제목, 제목 표시줄
	<meta charset="utf-8">	:문자 인코딩 방법

<body></body>	: 실제 브라우저에 표시될 내용
      --------------------------------------------	
			<header 헤더>
      --------------------------------------------
	<nav>	|	<section>      |<aside>
		|		       |
		|	 <article>     |
		|	 <article>     |
      --------------------------------------------
			<footer>
      --------------------------------------------	
	<header> 내용</header>		: 특정 부분의 머리말
	
	<iframe src="삽입할 문서 주소" [속성 = "속성 값"]></iframe>	: 외부문서 삽입	
		-iframe 태그 속성들-
		width	:인라인 프레임의 너비 (단위: 픽셀)
		height	:인라인 프레임의 높이 (단위: 픽셀)
		name	:인라인 프레임의 이름
		seamless:테두리 없앰, 속성값 없음
		src	:표시할 문서의 주소

	<article>내용</article>		:실제 콘텐츠 내용
	<aside>내용</aside>		:본문 이외의 내용(광고, 링크같은 사이드바)
	

	<footer>내용</footer>		:제작 정보와 저작권정보
		<address>내용</address>	:제작자 정보, 연락처 정보(우편주소x)
	

	<div id="이름"></div>		:해당 부분을 한 id로 선택할수 있게함(#이름{내용}으로 스타일 지정가능)

	
	<h숫자> 제목 </h숫자>		:제목 표시하는 용도 (1-6, 1이 가장큼)
	<p> 텍스트 한 단락</p>		:앞뒤로 빈줄생김, 줄바꿈 없으면 창너비에따라 자동 줄바꿈
	<br>				:줄바꿈
	
	<b>내용</b>			:강조x 진하기
	<strong>내용</strong>		:강조를 위한 진하기
	<i>내용</i>			:이텔릭체 강조x
	<em></em>			:이텔릭체 강조o
	<mark>내용</mark>		:형광펜효과
	<s>내용</s>			:취소선
	<sup>위첨자</sup>		:수학식에서 위첨자
	<sub>아래첨자</sub>		:수학식에서 아래첨자

	<!-- 주석 -->			:주석 (두줄 이상도됨)	

	<nav>메뉴</nav>			:메뉴 넣을때 사용 (nav는 header,footer,aside안에 들어가거나 독립적으로 사용 가능)
		<ul>목록</ul>	
		<ol>목록</ol>		:순서 목록
			type 속성	:키워드[1,a,A,i,I]
			start속성	:중간번호부터 시작가능 [start="중간번호"]
			reversed속성	:번호 역순 표시

	<dl>정의하는 목록</dl>		:사전같이 정의 - 설명 순으로 목록을 만드는거
		<dt>제목</dt>		:설명하고자 하는 제목
		<dd>설명</dd>		:제목을 설명
	
	<table></table>			:표를 만들기 위해 가장 먼저 써야할거
		<caption></caption>	:표에 간단한 제목(이거 안쓸거면  table의 summary 속성으로 summary="설명 내용" 써줘야함)
		<tr>내용</tr>		:행 추가
		<td>내용</td>		:셀 추가
		<th>내용</th>		:제목이 들어갈 셀 추가
			colspan="수"	:쓰인 곳부터 수만큼 행을 합침
			rowspan="수"	:쓰인 곳부터 수만큼 열을 합침
		<col>			:한 열에 있는 모든 셀 묶기(스타일 지정할 때 사용)
		<colgroup></colgroup>	:여러개의 col태그를 묶어 스타일을 적용하기위함(안묶고 span="수" 를 이용해서 그 지점부터 수 만큼의 열을 스타일 지정 가능)
		<thead></thead>		:표의 제목부분
		<tfoot></tfoot>		:표의 요약부분
		<tbody></tbody>		:표의 본문

	<pre>내용</pre>			:안에 쓰는 공백은 생략되지 않고 다 표시됨(프로그램 소스 작성할 때 사용)

	<q>인용문</q>			:인용을 위한 태그, 자동 따옴표""

	
	<time [datetime="값"]> 내용</time>	: 일반적으로 표시되지만 브라우저는 날짜라고 인식하도록 하기 위함
		datatime="YYYY-MM-DDThh:mm:ssZ"			(T: 시간구분, Z는 시간대)

	<span style="스타일">내용</span>:아무 의미 없지만 줄바꿈 없이 영역에 스타일 지정하기 위해 씀

	<figcaption>내용</figcaption>	:캡션 설명 붙이기(figure 태그 안에 씀)

	<form>폼 요소</form>
		***속성***
		method 	: 폼을 전송할 방식(get은 256~4096바이트, post는 제한 없음)을 고름
		name	: 여러폼을 식별하기 위한 것
		action	: 폼을 전송할 서버쪽 스크립트파일 지정. action말고 onsubmit이벤트를 이용해 스크립트로 처리 가능
		target	: action에서 지정한 스크립트 파일을 현재창이 아닌 다른 위치에 열도록 지정한다.
	   <fieldset></fieldset>
		disabled: fieldset의 자식요소들을 사용할 수 없게 만듬. 화면에는 표시되나 입력, 선택 못함
		form	: 현재<fieldset> 태그가 속해 있는 form의 이름 표시
		name	: 이 속성을 이용해 서버로 넘겨줄 이름을 지정 가능.
	   <label>
		*태그 안에 폼요소를 넣는 방법
		*<label [속성="속성 값"] for="이름">묶을 텍스트 내용</label> 으로 따로 떼어서 묶음
	<input></input>
		hidden	: 보이지는 않지만 서버로 넘겨지는 값을 가짐
		text	: 한줄짜리 텍스트를 입력할 수 있는 상자를 넣음
		search	: 검색 상자를 넣습니다.
		tel	: 전화번호 입력 필드를 넣습니다.
		url	: URL 주소를 입력할 수 있는 필드를 넣는다.
		email	: 메일 주소를 입력할 수 있는 필드를 넣는다.
		password: 비밀번호를 입력할 수 있는 필드를 넣는다.
		number	: 숫자를 조절할 수 있는 화살표를 넣는다.
		range	: 숫자를 조절할 수 있는 슬라이드 막대를 넣는다.
		color	: 색상표를 넣는다.
		checkbox: 주어진 항목에서 2개 이상 선택 가능한 체크박스를 넣는다.
		radio	: 주어진 항목에서 1개만 선택할 수 있는 라디오 버튼을 넣는다.
		submit	: 서버 전송 버튼을 넣는다.
		image	: submit버튼 대신 사용할 이미지를 넣는다.
		reset	: 리셋버튼을 넣는다.
